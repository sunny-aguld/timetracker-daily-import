# src/lib/workItems.ts 解説（初学者向け）

## このファイルの役割
ワークスペース内から `work_items.json` を見つけて読み込み、型チェックして返します。

## 主な型
- `WorkItem`: `{ name: string; id: string }`
- `WorkItemsByCategory`: カテゴリ名をキーにした配列マップ

## 処理の流れ
1. ワークスペースが開かれているか確認
2. 設定値 + 既定パスから候補パスを順に探索
3. ファイル読み込みと JSON パース
4. 各カテゴリと要素の型を検証
5. `absolutePath` と `data` を返す

## 初学者ポイント
- 設定値を優先しつつ、既定値にフォールバックする設計です。
- 先にバリデーションすることで、後続処理での実行時エラーを減らせます。
