# src/lib/tlParser.ts 解説（初学者向け）

## このファイルの役割
Markdown の `## TL` セクションを解析し、機械処理しやすい配列データに変換します。

## 何をパースするか
例:
- `09:00 @task=XXX #id=1234 メモ`
- `12:00 休憩`（marker として扱う）

## 主要関数
- `extractTlLines(markdown)`
  - `## TL` セクションだけ抜き出す
- `parseTlLinesStrict(lines)`
  - 各行を `time/body` に分解し、`work` または `marker` に分類

## 厳密チェック
- 時刻は `HH:MM` のみ許可
- 00:00〜23:59 の範囲チェック
- 形式不正時は行番号付きエラー

## 初学者ポイント
- まず「文字列を抽出」し、次に「意味を付ける」2段階で実装しています。
- `lineNo` を持たせると、ユーザーに修正箇所を伝えやすくなります。
