# instruction フォルダについて

このフォルダには、`src` 配下の `.ts` ファイルごとの解説書を置いています。  
対象読者は「TypeScript と VS Code 拡張機能の初学者」です。

## 読み方
1. `extension.ts.md` を最初に読む
2. 次に `commands` 系を読む
3. `lib` 系でデータ処理と API 連携を理解する
4. `providers` で補完の仕組みを理解する
5. `test` でテストの現状を確認する

## 対応方針
- 1ファイル = 1解説書（`.ts` ごと）
- 実装の「何をしているか」を優先
- 初学者が追いやすいように、難しい語は短く補足
